Woocommerce PostNL plugin

Inleiding

Deze handleiding beschrijft de functionaliteiten en de installatie van de PostNL plugin voor WooCommerce.

De meest belangrijke functionaliteiten zijn:

  • Eenvoudig zendingen voormelden bij PostNL en verzendlabels printen.
  • Bezorgopties in de check-out aanbieden (Ophalen bij een PostNL punt, Dagkeuze en Avondbezorging).
  • Retourlabels creëren.
  • Valideren van Nederlandse adressen.
  • Bestellingen als Brievenbuspakje versturen.

De plug-in kan zowel door Nederlandse als Belgische webwinkeliers gebruikt worden.

Let op, de plug-in is vernieuwd. Alle versies hoger dan v5.0.0. worden beschouwd als nieuw. Als je voor het eerst de vernieuwde plug-in installeert, dan dien je de plug-in opnieuw te configureren.

Inhoud

2 Installatie

Wij adviseren om de installatie van de plugin altijd eerst op een testomgeving uit te voeren en om altijd een back-up van je webshop te maken voordat je begint.

Je kunt de installatie op twee manieren uitvoeren:

  1. De plugin handmatig uploaden
  2. Via de WordPress plugin directory (vanaf 6 februari 2023)

2.1 Handmatig uploaden

Wilt u de PostNL plugin handmatig uploaden en installeren? Volg dan onderstaande stappen:

  1. Download de zip file van de plugin via deze site
  2. Klik rechtsboven op de groene knop Code → Download ZIP
  3. Ga in WordPress naar het menu Plugins → Nieuwe plugin.
  4. Klik bovenin het scherm op Uploaden.
  5. Kies het bestand en klik op Nu Installeren.
  6. Wanneer de download klaar is klikt u op Plugin activeren.

2.2 Via de WordPress plugin directory (vanaf 6 februari 2023 23:59)

Om de PostNL plugin te installeren via de WordPress plugin directory volgt u onderstaande stappen:

  1. Ga in WordPress naar het menu Plugins → Nieuwe plugin.
  2. Typ in de zoekbalk het woord PostNL en klik op zoeken.
  3. Kies de plugin PostNL for WooCommerce en klik op Nu installeren. De plugin wordt gedownload.
  4. Wanneer het downloaden voltooid is, klikt u op Plugin activeren.

3 Configuratie

Ga naar het menu WooCommerce → Instellingen → Verzendmethoden → PostNL om de PostNL plugin te configureren.

3.1 Account Instellingen

Modus omgeving

Je kunt de WooCommerce-omgeving koppelen aan de Sandbox- of Productie omgeving van PostNL. Let op, de Sandbox omgeving is een testomgeving. Hier kun je test-zending maken. Alleen zendingen die in de Productieomgeving zijn voorgemeld kun je inleveren bij PostNL. Een zending wordt in rekening gebracht zodra hij de eerste scan krijgt bij PostNL.

API Key

In je Mijn PostNL- omgeving vind je naast jouw klantgegevens ook je API keys. Voor de Sandbox- en Productieomgeving zijn twee verschillende API keys beschikbaar. Selecteer eerst de Sandbox-omgeving en voeg hier de Sandbox-API key toe. Sla de instellingen op en herhaal deze stappen voor de Productieomgeving.

Logging

Activeer de functionaliteit logging om makkelijker te kunnen achterhalen wat de oorzaak van verschillende technische issues is.

Klantgegevens

Vul jouw PostNL-klantnummer (bijv. 123456) en klantcode (bijv. ABCD) in. Deze gegevens vind je in jouw Mijn PostNL- omgeving.

3.2 Retour Instellingen

Altijd retour-label printen

Als je een verzendlabel voor een heenzending aanmaakt is het mogelijk om tegelijkertijd ook het verzendlabel voor de eventuele retourzending aan te maken en te printen. Een fijne service voor je klanten.

Antwoordnummer

Als Nederlandse klant ben je verplicht om een Antwoordnummer te gebruiken voor retour-zendingen. Dit geldt niet voor Belgische klanten.

Klantcode retouren

Let op, de klantcode Retouren is niet hetzelde als de klantcode die je bij ‘3.1. Account Instellingen’ hebt ingevuld. Jouw klantcode Retouren vind je in je Mijn PostNL- omgeving.

Retour-label per individuele order printen

Het is mogelijk om per order te bepalen of een retourlabel wel of niet direct gemaakt en geprint moet worden. Dit doe je door in een order het vinkje Creëer retourlabel wel of niet aan te vinken.

Als de functionaliteit Altijd retour-label printen met label heenzending aanstaat, staat dit vinkje automatisch ook in de order zelf aan.

3.3 Checkout Instellingen

Ophalen bij een PostNL-punt

Door de functionaliteit Ophalen bij een PostNL-punt te activeren kunnen jouw klanten in de checkout kiezen om hun bestelling op te halen bij een PostNL-punt i.p.v. de bestelling thuis te ontvangen. Met deze optie vergroot je de controle van jouw klanten over hun bestelling.

Bezorgdagen & Aantal bezorgdagen

Door Bezorgdagen te activeren kunnen jouw klanten in de checkout bepalen op welke dag zij hun pakket thuis ontvangen. Wist je dat het aanbieden van bezorgopties conversie-verhogend kan werken? Probeer het nu zelf.

Jij bepaalt hoeveel bezorgdagen er in de checkout zichtbaar zijn. Het aantal dagen kan variëren tussen 1 en 12 dagen.

Let op, er moet rekening met de gekozen bezorgdag gehouden worden bij het overhandigen van de pakketten aan PostNL. Kiest de consument om volgende week donderdag de bestelling te ontvangen, dan dien jij op woensdag het pakket aan PostNL te overhandigen (uitgaande van een overkomstduur van 1 dag).

Ochtend bezorging & Toeslag

Voor klanten die hun pakket alleen overdag in ontvangst kunnen nemen kan je gebruik maken van ochtend bezorging. Door Ochtend bezorging te activeren geef je jouw klanten deze mogelijkheid. De kosten voor ochtend bezorging liggen hoger. Jij kunt een deel van deze kosten doorberekenen aan jouw klanten door een toeslag voor ochtend bezorging te vragen.

Avondbezorging & Toeslag

Consumenten die overdag niet huis zijn geven de voorkeur om ’s avonds hun bestelling te ontvangen. Door Avondbezorging te activeren geef je jouw klanten deze mogelijkheid. De kosten voor een avondzending liggen hoger. Jij kunt een deel van deze hogere kosten doorberekenen aan jouw klanten door een toeslag voor avondbezorging te vragen.

Overkomstduur

Met de overkomstduur geef je aan hoeveel dagen je nodig hebt om de bestelling gereed te maken. Stel dat er vandaag een bestelling wordt geplaatst die je morgen al kan bezorgen, dan is de overkomstduur 1.

Cut off tijd

De cut off tijd geeft aan tot wel tijdstip een bestelling geplaatst kan worden zodat de overkomstduur gehaald kan worden. Voor bestellingen die na de cut off tijd besteld worden, wordt er één dag bij de overkomstduur opgeteld.

Verzenddagen

Geef aan op welke dagen je de pakketten aan PostNL overhandigd. Deze gegevens zijn nodig om het eerst volgende bezorgmoment te kunnen bepalen.

Overhandig jij ook op zondag pakketten aan PostNL? Zorg dan dat het contractueel is vastgelegd dat PostNL jouw pakketten op zondag sorteert. Alleen dan kunnen jouw klanten op maandag hun bestelling ontvangen.

Valideer Nederlandse adressen & Gebruik PostNL adresvelden

Om te voorkomen dat foutieve adressen worden ingevoerd, kan de adrescheck geactiveerd worden. Deze adrescheck valideert Nederlandse adressen. Als het adres niet bekend is wordt de consument gewaarschuwd zodat hij/zij het adres kan aanpassen.

Het is aan te raden om de PostNL adresvelden te activeren. In de checkout worden dan aparte velden getoond voor de straatnaam, huisnummer en huisnummertoevoeging. Aan de hand van deze velden is het makkelijker om de adressen te valideren.

3.4 Versturen buiten Europa

Globalpack Barcode Type en Globalpack Klantcode

Voor het versturen van pakketten buiten Europa (Globalpack zendingen) zijn extra klantgegevens nodig. Deze gegevens zijn nodig om de juiste barcode te genereren. Vul jouw Globalpack Barcode Type en Globalpack Klantcode in. Deze gegevens zijn terug te vinden in jouw Zakelijke PostNL omgeving.

GS tarief code en land van herkomst

De GS tarief code en land van herkomst is benodigde informatie voor de douaneformulieren. Deze gegevens dienen voor elk product dat in jouw webshop wordt aangeboden beschikbaar te zijn.

Ga naar Producten → Open een product → Productgegevens – Verzendmethoden en voer het land van herkomst en de GS tarief code in. Herhaal dit voor elk product.

Is de GS tarief code en het land van herkomst voor bijna alle producten hetzelfde? Vul dan bij de algemene PostNL instellingen een Standaard GS tarief code en een Standaard land van herkomst in. Deze gegevens worden overgenomen voor de douaneformulieren als er op productniveau geen GS tarief code of land van herkomst bekend is.

3.5 Printer en Email Instellingen

Labelformaat

Welke type printer gebruik je om labels te printen? Gebruik je een labelprinter selecteer dan A6 labelformaat. Selecteer labelformaat A4 als je gebruik maak van een reguliere printer.

WooCommerce Email en Tekst

Wil je jouw klanten vanuit WooCommerce een Track & Trace mail sturen? Zet het vinkje achter WooCommerce Email dan aan. In het WooCommerce Email Tekst veld kan je aangeven welke tekst er in deze mail word meegestuurd. Onze tekstsuggestie is "Gebruik deze link om jouw zending te volgen: {tracking-link}"

4 Verzendklassen en verzendprijs

Met de installatie van de PostNL plug-in is ook een PostNL Verzendmethode aangemaakt. Deze verzendmethode koppelen je aan de WooCommerce Verzendzones. Per verzendzones kan je de verzendprijs bepalen.

5 Orders verwerken

Orderoverzicht

In het orderoverzicht is te zien voor welke bestelling gekozen is voor verzending met PostNL. Deze bestellingen zijn te herkennen doordat via PostNL onder de bestemming staat. In de PostNL Tracking kolom is de barcode te zien van de zending van deze order. Dit is ook een link naar de consumenten Track & Trace pagina van PostNL.

Orderpagina

Open een order. Rechts boven op de orderpagina is het PostNL menu te zien. Hierin is te zien om welke Type bezorging het gaat; standaard bezorging, ophalen bij een PostNL-punt, Avonbezorging, Europese bezorging of Wereldzending. Ook is de ‘Bezorgdatum’ van de bestelling te zien. Let op, houdt rekening met deze bezorgdatum om te bepalen wanneer de pakketten aan PostNL overhandigd moeten worden.

Extra verzendopties

Het is mogelijk om een extra verzendoptie toe te voegen aan de PostNL zending.

Afhankelijk van het type bezorging is het mogelijk om de zending:

  • te verzekeren (verzekerd bedrag is totale verzendbedrag)
  • retour te sturen als ontvanger niet thuis is
  • handtekening voor ontvangst te vragen
  • het pakket alleen bij de ontvanger te bezorger (niet bij buren)
  • of het type bezorging te wijzigen naar Brievenbuspakje.

Aantal labels

In het PostNL menu is het mogelijk om het aantal labels van een zending te wijzigen. Zo kan het bijvoorbeeld voorkomen dat een zending opgedeeld moet worden naar twee pakketten omdat de zending anders te groot of te zwaar is. Verander het aantal labels in dit geval naar 2.

Creëer retourlabel

Bij de Retour Instellingen is aangegeven of er wel/niet altijd een retour-label geprint moet worden bij het aanmaken van het heenzending. Deze instelling wordt in dit menu overgenomen. Aan de hand van het ‘Creëer retourlabel’ vinkje is het mogelijk om de Retour Instellingen te overschrijven.

Zending aanmaken en verzendlabel printen

Staan alle vinkjes in het PostNL menu zoals gewenst? Klik dan onderaan in het menu op ‘Zending aanmaken’. Met deze actie wordt de zending voormeld bij PostNL. Klik vervolgens op ‘Print Label’ om het verzendlabel te downloaden.

Meerdere Labels tegelijk aanmaken

Om tijd te besparen is het handig om meerdere labels tegelijk te kunnen printen. Selecteer in het orderoverzicht de bestellingen waarvoor je een label wil printen → klik op Bulkacties → ‘PostNL Create Label’.

Selecteer vervolgens in het PostNL menu of je nog extra verzendopties wil toevoegen → Klik op Submit → Klik op Download file → De verzendlabels van de verschillende orders worden in één document gedownload.
